# 🏥 DoctorTime Servlets

A Java Web Application built with **Servlets** for managing medical examinations between patients and doctors.  
The system supports user registration, authentication, messaging and examination management using a **PostgreSQL database** with **JDBC**.

---

## 📌 Project Description

DoctorTime Servlets is a web-based system that simulates a medical platform where **patients** and **doctors** can interact.

The application is built using:
- HTML for the frontend
- Java Servlets for backend logic
- PostgreSQL for database management
- JDBC for database connectivity
- Apache Tomcat as the application server

---

## ✨ Features

### 👤 User Management
- User registration (Patient or Doctor)
- Secure login system (email, password, role selection)
- Session handling

---

### 🧑‍⚕️ Patient Features
- View medical examination history
- View list of all registered doctors
- Send messages to doctors requesting examinations
- Delete account and all related data
- Logout functionality

---

### 🩺 Doctor Features
- View examinations assigned to their patients
- Add new medical examinations for patients
- View messages from patients
- Delete account and all related data
- Logout functionality

---

### 🌐 General Features
- Public homepage with medical articles (embedded external websites)
- Information page about the application
- Navigation menu available for all users

---
